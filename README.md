## Getting started

#### Requirements 

```javascript

1. Install MongoDB:
Download it from here: https://docs.mongodb.com/manual/administration/install-community/

2. Run Backend
$ npm install
$ npm start

3. Run Frontend
*open new terminal*
$ cd frontend
$ npm install
$ npm start

```
